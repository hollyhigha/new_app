# 占位图目录

本目录下的图片为**首版开发期占位图**，上架前必须逐一替换为**自有版权或已获授权**的图片，
否则可能因"图片版权问题"被应用商店拒审（尤其是华为 / 应用宝 / 小米 医美类专审）。

## 必须替换清单

| 文件 | 当前状态 | 使用位置 | 建议规格 |
|---|---|---|---|
| `article-cover.png` | 占位图 | 文章/问答/贴士/案例封面 | 750×420 / JPG/PNG |
| `banner-1.png` ~ `banner-3.png` | 占位图 | 首页轮播 | 750×340 / JPG |

## 版权来源建议

1. **付费商用**：视觉中国（~30 元/张）、图虫 Plus、站酷海洛
2. **免费商用**：Unsplash / Pexels（注意：**涉及真人面部照需确认有"肖像使用授权"**，否则触红线）
3. **自制插画**：Midjourney / 文心一格 生成非写实插画，**规避 AI 真人照**（医美类绝对不能用）

## 图片内容限制

- ❌ 禁止"治疗前后对比图"（《医疗广告管理办法》第七条）
- ❌ 禁止真实医师 / 机构的宣传照
- ❌ 禁止药品、器械包装特写
- ✅ 推荐：科普向图示、护肤场景、卡通插画
