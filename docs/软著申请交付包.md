## 四、软件功能说明（申请表填写稿）

### 4.1 软件简要说明（**300 字以内，申请表原文**）

> 靓人科普医美资讯服务软件 V1.0（以下简称"本软件"）是一款面向普通用户的医学美容科普信息展示类移动应用软件。本软件基于 uni-app 跨端框架开发，支持 Android 与 iOS 双端运行，后端采用 uniCloud 阿里云 Serverless 架构，实现了科普文章浏览、分类检索、搜索推荐、收藏历史、账号登录、在线留言、意见反馈及合规隐私保护等核心功能。
>
> 本软件通过云函数服务提供安全、稳定、低延迟的数据访问能力，结合本地缓存与增量拉取策略提升访问体验；通过隐私弹窗、权限最小化、未登录浏览等机制满足工信部《App 违法违规收集使用个人信息行为认定方法》及国家《网络安全法》《个人信息保护法》等合规要求。
>
> 本软件仅提供医学美容领域的科普信息展示与用户互动，不提供诊疗服务、不销售药品器械、不展示医疗广告内容。

### 5.2 技术亮点（可直接写入"软件特点"一栏）

1. **跨平台一次开发，多端部署**：基于 uni-app + Vue 3 Composition API，同一套业务代码同时运行于 Android、iOS，显著降低开发与运维成本。
2. **Serverless 后端架构**：采用 uniCloud-aliyun 云函数，按调用量计费、免运维、弹性扩缩容，天然适配医美科普类应用的间歇性流量。
3. **隐私合规设计**：内置隐私弹窗组件（`components/privacy-popup`）、权限最小化（仅申请网络相关权限）、未登录即可浏览、敏感操作前显式二次确认，满足工信部合规要求。
4. **医美类内容风险防控**：顶部常驻免责声明"本平台内容仅供科普参考，不构成诊疗建议"、禁用疗效承诺、内容与审核双重校验。
5. **离线降级能力**：`utils/mock-data.js` 在云端不可用时提供兜底数据，保障基础浏览体验。
6. **转化回传可靠性**：`report-conversion` + `retry-conversion` 双管齐下，巨量引擎事件上报失败后自动重试，提升投放 ROI。
7. **运营实时触达**：`notify-wechat` 云函数将关键业务事件（留言、注册、异常）实时推送至企业微信群，客服可秒级跟进。
8. **多级内容结构**：articles / qa / tips 三张表支撑"文章 / 问答 / 小贴士"三种内容形态，通过统一的 `get-content` 云函数聚合下发。

## 附录 B：风险提示

1. **主体风险**：软著著作权人一经登记，变更需走"著作权转让"流程（30 工作日或加急 1000~2500 元）。若公司主体未定，建议先暂缓。
2. **发表日期风险**：`首次发表日期`一旦填写即不可撤回。未上架前务必填 **"未发表"**，避免与商店提交时间冲突引发审核质疑。
3. **软件名称风险**：软著名称建议与商标名称保持一致。若"靓人科普"未来计划注册商标，建议在递交软著前先完成**商标检索**（0~200 元，1 小时）。
4. **分类风险**：若填写"医疗类"软件，部分审核员会要求补交《医疗器械软件注册证》，本应用为**科普信息类**，请代理明确申报为"应用软件 / 移动应用软件"，行业类别填"医疗健康信息服务"而非"医疗软件"。
5. **代码合规风险**：严格按 §6.1 白名单抽取，`uni_modules/` 第三方代码一旦混入，被抽查后会被认定"抄袭"并驳回。
6. **脱敏风险**：源代码中如残留 AccessKey 等敏感信息，一旦证书在网上公示（CPCC 可查），将直接泄露生产密钥，需**自查三遍**。
7. **加急风险**：加急件审查速度快但仍可能补正；补正期间工期重新计算，务必预留 5 个工作日缓冲。
